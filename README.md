Athena App

## Running the app

```bash
# development With Build
$ docker-compose -f docker-compose.dev.yaml up --build

# development With out Build
$ docker-compose -f docker-compose.dev.yaml up

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
